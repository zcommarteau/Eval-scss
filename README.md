# Evaluation scss Zakaria Commarteau

## Le html est la structure de notre page web, le css le style et le scss permet d'alléger le css. Trop de css peut devenir illisible et donc le scss intervient. C'est un outil qui permet de rajouter du style, des animations, du responsive à notre page web.



## La méthode BEM est issue de l'acronyme Blocks Elements Modifiers. Le bloc est le composant parent qui peut contenir un ou plusieurs éléments. Puis l'élément est le composant qui appartient à un bloc c'est comme l'enfant d'un bloc. Enfin, un modificateur est une classe css qui apporte la notion de comportement. Le modificateur s'applique aux blocs et aux éléments.



## Un prépocesseur en scss permet d'augmenter les fonctionnalités de base du css avec son propre langage. On peut faire des apports de variables.



## Les variables en scss peuvent prendre n'importe quelles valeurs css valide. Il faut déclarer les variables dans la classe :root



## Les mixins sont comme des templates. Elles peuvent êtres définies en haut d'une page scss et réutiliser par la suite. C'est un gain de temps et cela permet d'alléger la lisibilté du code.
